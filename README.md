# computer-vision
컴퓨터 비전 모델 학습을 위한 이미지 데이터 전처리 및 학습 파이프라인

이미지 데이터 전처리 및 컴퓨터 비전 실험을 위한 Jupyter 노트북과 전처리 스크립트를 제공하는 레포지토리

주요 내용

이미지 데이터 전처리 파이프라인(스크립트)
실험 기록 및 예제 코드(Jupyter 노트북)
레포지토리 구성

README.md — 프로젝트 설명 및 사용법 (이 파일)
computer_vision.ipynb — 실험/분석을 수행하는 Jupyter 노트북 
image data preprocessing.py — 이미지 데이터 전처리를 위한 Python 스크립트 
기타(레포지토리 루트) — 추가 자료 및 리소스 링크
설치 및 요구사항

Python 3.8+ 권장
주요 라이브러리(예시): numpy, pandas, opencv-python, pillow, matplotlib, scikit-image, torch/tensorflow(모델 사용 시)
Colab 사용 권장: 별도 환경 설정 없이 notebook을 바로 실행하여 실험 가능
